Guia Completo de Flexbox

Categoria: CSS Publicado em: 16/06/2025

Neste tutorial, vamos explorar o CSS Flexbox, um modelo de layout poderoso que permite criar designs complexos e responsivos com muito menos código e de forma mais intuitiva.

O que é Flexbox?

O Flexible Box Layout, ou simplesmente Flexbox, é um modelo de layout unidimensional que oferece uma maneira eficiente de alinhar e distribuir espaço entre itens em um container, mesmo quando o tamanho desses itens é desconhecido ou dinâmico.

Propriedades do Container Flex

Para começar, você precisa aplicar `display: flex;` ao container pai. Algumas propriedades importantes do container são:

  • flex-direction: define a direção do eixo principal (row, column, etc.).
  • justify-content: alinha os itens ao longo do eixo principal.
  • align-items: alinha os itens ao longo do eixo transversal.
.container {
    display: flex;
    justify-content: center;
    align-items: center;
}
